The `ChromaHandshake` class provides seamless integration between Chonkie's chunking system and ChromaDB, a popular vector database. 

Embed and store your Chonkie chunks in ChromaDB without ever leaving the Chonkie SDK.


## Installation

Before using the Chroma handshake, make sure to install the required dependencies:

```bash
pip install chonkie[chroma]
```

## Basic Usage

### Initialization

```python
from chonkie import ChromaHandshake

# Initialize with default settings (in-memory ChromaDB)
handshake = ChromaHandshake()

# Or specify a persistent storage path
handshake = ChromaHandshake(path="./chroma_db")

# Or use an existing Chroma client
import chromadb
client = chromadb.Client()
handshake = ChromaHandshake(client=client, collection_name="my_collection")
```

### Writing Chunks to ChromaDB

```python
from chonkie import ChromaHandshake, Chonker

handshake = ChromaHandshake() # Initializes a new Chroma client

chunker = Chonker()
chunks = chunker("Chonkie is the best chonker ever!")

# Now, we write the chunks to the Chroma collection
handshake.write(chunks)
```

### Searching for Chunks

After writing chunks to the Chroma collection, you can use the `search` method to find the most relevant chunks to a given query.

```python
from chonkie import ChromaHandshake, Chonker

handshake = ChromaHandshake()

# First, let's write some chunks to the collection
chunker = Chonker()

# Write some chunks to the collection
text_to_chunk = "Chonkie is a library for chunking text. It is a simple and easy to use library that can be used to chunk text into smaller pieces. It is a great tool for NLP tasks."

chunks = chunker(text_to_chunk)

handshake.write(chunks)

# Now, we can search for the most relevant chunks
results = handshake.search(query_texts=["What is Chonkie?"], n_results=1)

# The results are a dictionary containing the metadatas, documents, and distances
print(results)
```
