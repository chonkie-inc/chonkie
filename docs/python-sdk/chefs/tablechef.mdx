The `TableChef` is a chef that extracts tables from markdown text and prepares them for future usage. It parses markdown tables and converts them into a structured format (list of rows).

## Usage

<CodeGroup>

```python markdown
from chonkie import TableChef

md_tables = """
# my table md

## table 1

| Name | Age | City |
|------|-----|------|
| Alice | 25 | New York |
| Bob | 30 | Paris |
| Charlie | 35 | London |

tables are great

## table 2
| Product | Price | Quantity |
|---------|-------|----------|
| Laptop | 1000 | 5 |
| Phone | 500 | 10 |

"""
table_chef = TableChef()
tables = table_chef.process(md_tables)
print(tables)
```

```python csv/excel
from chonkie import TableChef
table_chef = TableChef()
table = table_chef("my_table.csv") 
```
