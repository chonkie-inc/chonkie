---
title: "Model2VecEmbeddings"
description: "Embed text using Model2Vec embeddings"
---

Embeddings are handled by the `Model2VecEmbeddings` class, which is a wrapper around the `model2vec` library.

## Installation

Embeddings require the `model2vec` library. See the [Installation Guide](/chonkie/installation) for more information.

## Usage

```python
from chonkie import Model2VecEmbeddings

embeddings = Model2VecEmbeddings()
```
