---
title: "SentenceTransformerEmbeddings"
description: "Embed text using SentenceTransformer embedding models"
---

Embeddings are handled by the `SentenceTransformer` class, which is a wrapper around the `sentence-transformers` library.

## Installation

Embeddings require the `sentence-transformers` library. See the [Installation Guide](/chonkie/installation) for more information.

## Usage

```python
from chonkie import SentenceTransformerEmbeddings

embeddings = SentenceTransformerEmbeddings("all-MiniLM-L6-v2")
```
