description: "Embed text using OpenAI embeddings"
---

Embeddings are handled by the `OpenAIEmbeddings` class, which is a wrapper around the `openai` library.

## Installation

Embeddings require the `openai` library. See the [Installation Guide](/chonkie/installation) for more information.

## Usage

```python
from chonkie import OpenAIEmbeddings

embeddings = OpenAIEmbeddings()
```
