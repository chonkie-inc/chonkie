Chonkie Parse is our custom-trained OCR model that extracts clean text from documents, images, and PDFs. It's designed to work seamlessly with Chonkie's text chunking pipeline.

<Note>
  Chonkie Parse is currently in **private alpha** and is only available through
  Chonkie Cloud.
</Note>

## Key Features

- Extract text from PDFs, images, and scanned documents
- Custom-trained model optimized for document processing
- Seamless integration with Chonkie chunkers
- Available through Chonkie Cloud API

## Access

Chonkie Parse is currently available to select users in private alpha. Contact us at [support@chonkie.ai](mailto:support@chonkie.ai) for access.
