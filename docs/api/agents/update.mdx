---
title: "Update Agent"
api: "PUT https://labs.chonkie.ai/api/v1/agents/{agent_slug}"
description: "Modify an agent's configuration"
---

Update an existing agent's configuration.

## Request

#### Path Parameters

<ParamField path="agent_slug" type="string" required>
  The slug of the agent to update.
</ParamField>

#### Body Parameters

All parameters are optional. Only include fields you want to update.

<ParamField path="name" type="string">
  New agent name.
</ParamField>

<ParamField path="description" type="string">
  New description.
</ParamField>

<ParamField path="model" type="string">
  Change the LLM model.
</ParamField>

<ParamField path="systemPrompt" type="string">
  Update system prompt.
</ParamField>

<ParamField path="temperature" type="float">
  Adjust temperature (0.0-2.0).
</ParamField>

<ParamField path="maxContextChunks" type="integer">
  Change max context chunks.
</ParamField>

<ParamField path="useContext" type="boolean">
  Enable/disable context injection.
</ParamField>

<ParamField path="knowledge" type="string">
  Change knowledge base.
</ParamField>

## Response

#### Returns

<ResponseField name="agent" type="object">
  Updated agent configuration.
</ResponseField>

## Examples

<CodeGroup>

```python Python
import requests

agent_slug = "documentation-assistant"
url = f"https://labs.chonkie.ai/api/v1/agents/{agent_slug}"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
    "model": "gpt-4o",
    "temperature": 0.5
}

response = requests.put(url, headers=headers, json=data)
result = response.json()

print(f"Updated agent: {result['agent']['name']}")
print(f"New model: {result['agent']['model']}")
```

```javascript JavaScript
const agentSlug = 'documentation-assistant';
const response = await fetch(
  `https://labs.chonkie.ai/api/v1/agents/${agentSlug}`,
  {
    method: 'PUT',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      model: 'gpt-4o',
      temperature: 0.5
    })
  }
);

const result = await response.json();
console.log(`Updated agent: ${result.agent.name}`);
```
