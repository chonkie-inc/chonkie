---
title: "Deep Research"
api: "POST https://labs.chonkie.ai/api/v1/research"
description: "Create comprehensive research reports using AI agents"
---

Deep Research enables agents to conduct thorough, multi-step research on topics, generating comprehensive reports.

## Request

#### Parameters

<ParamField path="agent" type="string" required>
  Agent slug to use for research.
</ParamField>

<ParamField path="query" type="string" required>
  Research question or topic.
</ParamField>

## Response

#### Returns

<ResponseField name="reportId" type="string">
  Unique report identifier.
</ResponseField>

<ResponseField name="agentSlug" type="string">
  Agent slug used for research.
</ResponseField>

<ResponseField name="query" type="string">
  The research query.
</ResponseField>

<ResponseField name="status" type="string">
  Processing status.
</ResponseField>

<ResponseField name="createdAt" type="string">
  Creation timestamp.
</ResponseField>

## Examples

<CodeGroup>

```python Python
import requests

url = "https://labs.chonkie.ai/api/v1/research"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

data = {
    "agent": "documentation-assistant",
    "query": "What are the best practices for API authentication?"
}

response = requests.post(url, headers=headers, json=data)
result = response.json()

print(f"Research task created: {result['reportId']}")
print("Processing in background...")
```

```javascript JavaScript
const response = await fetch('https://labs.chonkie.ai/api/v1/research', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    agent: 'documentation-assistant',
    query: 'What are the best practices for API authentication?'
  })
});

const result = await response.json();
console.log(`Research task created: ${result.reportId}`);
```

```bash cURL
curl -X POST https://labs.chonkie.ai/api/v1/research \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "agent": "documentation-assistant",
    "query": "What are the best practices for API authentication?"
  }'
```

</CodeGroup>

## Get Research Report

Retrieve a completed research report:

```
GET /v1/research/{agent_slug}/{report_id}
```

## List Research Reports

List all research reports for an agent:

```
GET /v1/research/{agent_slug}
```

## Delete Research Report

Delete a research report:

```
DELETE /v1/research/{agent_slug}/{report_id}
```
