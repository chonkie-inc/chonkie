title: "List Agents"
api: "GET /v1/agents"
description: "Get all agents in your organization"
---

Retrieve a list of all AI agents in your organization, including their configuration and status.

## Authentication

Include your API key in the Authorization header:

```bash
Authorization: Bearer YOUR_API_KEY
```

Get your API key from [labs.chonkie.ai](https://labs.chonkie.ai)

## Request

This endpoint takes no parameters. Simply make a GET request with your API key.

## Response

#### Returns

Array of agent objects with full configuration.

## Examples

<CodeGroup>

```python Python
import requests

url = "https://labs.chonkie.ai/api/v1/agents"
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}

response = requests.get(url, headers=headers)
agents = response.json()

print(f"Found {len(agents)} agents:\n")
for agent in agents:
    print(f"- {agent['name']} ({agent['slug']})")
    print(f"  Model: {agent['model']}")
    print(f"  Context: {agent['useContext']}")
    if agent['useContext']:
        print(f"  Knowledge Base: {agent['knowledge']}")
    print()
```

```javascript JavaScript

});
```

## Filtering and Sorting

Filter agents in your application:

```python
def filter_agents_by_model(agents, model):
    """Get agents using a specific model."""
    return [a for a in agents if a['model'] == model]

def filter_agents_with_context(agents):
    """Get agents that use knowledge base context."""
    return [a for a in agents if a['useContext']]

def sort_agents_by_date(agents, reverse=True):
    """Sort agents by creation date."""
    return sorted(agents, key=lambda x: x['createdAt'], reverse=reverse)

# Usage
agents = response.json()
gpt4_agents = filter_agents_by_model(agents, "gpt-4o")
context_agents = filter_agents_with_context(agents)
recent_agents = sort_agents_by_date(agents)
```

## Use Cases

- **Dashboard** - Display all agents in admin panel
- **Selection** - Let users choose which agent to chat with
- **Monitoring** - Track agent usage and configuration
- **Audit** - Review which agents exist and their settings

## Best Practices

<Tip>
  Cache the agent list in your application and refresh periodically, rather than fetching on every interaction.
</Tip>

<Info>
  Agents are returned sorted by creation date (newest first) by default.
</Info>
